STAT CORRECTIONS STANDALONE

This is separate from PeaceKeys.

Do not run:
npm run dev

That command starts the PeaceKeys app, not this tool.

Use this instead:

1. Open the file named index.html in this folder
or
2. Double-click start-stat-corrections.bat

Files in this folder:

- index.html
- start-stat-corrections.bat

What to do:

1. Open the tool
2. Click "Load Sample" first
3. Paste StatBroadcast data on the left
4. Paste Analyzer data on the right
5. Click "Compare Data"

No terminal is needed for normal use.
